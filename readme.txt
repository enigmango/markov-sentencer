Run from command line with Python 2.7.6 or higher. Python 3 or higher may not run do to raw_input() function.

> markov.py <pathToTextFile> <anotherPathToTextFile>...
(multiple test files can be specified in one line)

Parameters: Any additional parameters are assumed to point to text files. Any number of input text files can be used. An asterisk as the first parameter will prompt for a custom seed, prefix length, and number of sentences. Any of these can be left blank to use the default.
